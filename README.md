# Мой трекер

Мини-приложение для ежедневного планирования, рабочих задач, денег, проектов и итогов.

## Как открыть локально

Открой файл `index.html` в браузере.

## Как опубликовать на GitHub Pages

1. Создай новый репозиторий на GitHub, например `my-tracker`.
2. Загрузи в него файлы из этой папки: `index.html`, `styles.css`, `app.js`, `README.md`, `.nojekyll`.
3. Открой `Settings` -> `Pages`.
4. В `Build and deployment` выбери `Deploy from a branch`.
5. В `Branch` выбери `main` и папку `/root`.
6. Сохрани настройки и подожди пару минут.

После этого GitHub даст ссылку вида:

```text
https://username.github.io/my-tracker/
```

## Как открыть с телефона

После публикации на GitHub Pages открой ссылку на телефоне. Можно добавить страницу на экран "Домой", и она будет ощущаться почти как маленькое приложение.

Важно: данные хранятся в браузере. То, что внесено на компьютере, не появится автоматически на телефоне. Для переноса используй экспорт и импорт JSON-файла в разделе "Итоги" -> "Резервная копия".

## Что отслеживать

- задачи на день с галочками;
- задачи по проектам: YouTour, Секта, дизайн и другое;
- время у задач, если нужно отметить созвон или слот;
- доходы по SMM и веб-дизайну отдельно;
- общие расходы;
- накопления по целям;
- проекты и их типы;
- итоги недели и месяца.

## Шрифт

В стилях первым указан `Gilroy`. Если Gilroy установлен в системе, браузер подхватит его автоматически. Если нет, будет использован системный fallback. Чтобы шрифт работал гарантированно на всех устройствах, нужно добавить файлы Gilroy в формате `.woff2` рядом с приложением и подключить их через `@font-face`.

## Где хранятся данные

Данные сохраняются локально в браузере через `localStorage`. Они не отправляются на GitHub и не видны другим людям.

Если другой человек откроет ссылку, у него будет своя отдельная версия трекера в его браузере.

## Базовая логика

Главная мысль приложения: сохранить простоту заметок с галочками, но добавить полезные автоматические подсчеты: сколько задач было по SMM и дизайну, сколько закрыто, сколько заработано, потрачено и отложено.
